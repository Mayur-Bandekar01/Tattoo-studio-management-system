# Technical Stack

## Backend
- **Language**: Python 3.12+
- **Framework**: Flask
- **Database**: MySQL (via `mysql-connector-python`)
- **Key Libraries**:
  - `Flask-Mail`: For email notifications.
  - `python-dotenv`: Mapping environment variables.
  - `mysql-connector-python`: Database connectivity.

## Frontend
- **Templating**: Jinja2
- **Structure**: Semantic HTML5
- **Styling**: Vanilla CSS (Modern design patterns, dark/light modes)
- **Interactions**: Vanilla JavaScript

## Infrastructure & Tools
- **Environment Management**: `.env` files
- **Linting**: Ruff (`.ruff_cache` present)
- **Deployment**: Traditional web server (running `app.py`)
